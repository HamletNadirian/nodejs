# Кроки запуску програми
#### 1. Встановіть залежності
Виконайте команду для встановлення всіх необхідних залежностей:
`npm install
`
#### 2. Запустіть програму з заповненням бази даних та тестами
   Виконайте команду для запуску програми, заповнення бази даних та тестів:
`npm run start:with-seed-test
`
#### 3. Дочекайтеся завершення процесу
   Після успішного виконання команди, база даних буде заповнена, тести виконані, а програма запущена.
   Після успішного запуску програми ви можете відкрити інтерфейс Swagger за наступним посиланням у вашому браузері:
   [http://localhost:7777/api-docs/](http://localhost:7777/api-docs/
)

#### 4 Налаштування програми
Файл .env **вже налаштовано**:
```HOST=localhost
HOST=localhost
PORT=7777
MONGO_ADDRESS=mongodb://localhost:27017/movie_rating_db
SEED_DATABASE=true
```
###### Примітка!
Під час **оновлення або створення сутності** (наприклад, фільму) вказуйте лише реальні та існуючі **ID** з бази даних.
Приклади **ID** у документації – умовні! Вони наведені лише для демонстрації структури запиту.
Де взяти дійсні **ID**?
Отримайте список сутностей із реальними **ID** за адресою:
http://localhost:7777/api-docs/#/Movies/get_api_movies
